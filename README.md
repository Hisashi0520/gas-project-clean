\# 月報管理システム（GAS）



\## 概要

Google Apps Script（GAS）を用いた月報管理システム。

スプレッドシートのデータをもとに、指定期間の集計結果を

Web UI で表示する。



\## 主な機能

\- 月次／任意期間での集計

\- Web UI による期間指定

\- 集計結果の即時表示



\## 構成

\- Code.js：集計・ビジネスロジック

\- Setup.js：初期設定・定義

\- index.html：画面UI

\- js.html / css.html：フロントエンド制御
